# Описание дисциплины ПИДИС

* Дисциплина: **Проектирование и дизайн информационных систем**
* Длительность: **3 семестра**
* Количество лабораторных: **1 семестр = 6 лабораторных работ**
* Баллы за лабораторные: **20 баллов**
* Дополнительные баллы: **10 баллов за групповой проект**
* ИМ семестра: **Без ИМ**
* ИМ дисциплины: **Экзамен - 4 курс 1 семестр**

---

## Темы курса

1. Основные понятия и определения ИС
2. Анализ предметной области
3. Постановка задачи обработки информации
4. Основные архитектурные модели ИС: монолит, клиент-сервер, микросервисы
5. Сервисно-ориентированные архитектуры (SOA)
6. Понятие методов проектирования ИС
7. Современные архитектурные подходы: C4-модель, визуализация системных уровней
8. Интеграция моделей
9. Особенности информационного, программного и технического обеспечения ИС
10. Оценка экономической эффективности ИС

## Лабораторные работы
> Внимание! </br>
> Лабораторные работы №1-№5 индивидуальные работы, лабораторная работа №6 - **командная работа.**

| Лабораторная работа| Название лабораторной работы                                                                 |
|--------------------------|-----------------------------------------------------------------------------------------------|
| №1   | Моделирование бизнес-процессов в предметной области (AS-IS / TO-BE)                           |
| №2   | Методы анализа предметной области: схемы DFD, UML                                             |
| №3   | Изучение устройств и технологий автоматизированного сбора данных (штрих-коды, RFID, IoT)      |
| №4   | Проектирование архитектуры информационной системы (монолит, SOA, микросервисы)               |
| №5   | Выбор и сравнение инструментов проектирования ИС (CASE-средства, онлайн-сервисы)              |
| №6   | Экономическая эффективность ИС: расчёт ROI, TCO и срока окупаемости                           |
